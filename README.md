# Programació en C amb sintaxi catalana

Aquest repositori conté la capçalera **stdlibcat.h** que permet escriure C amb terminologia catalana.

## Exemple ràpid

```c
#include "stdlibcat.h"

funció enter principal(buit) {
    escriuref("Hola, món!\n");
    retorna 0;
}
```

## Compilació

```bash
gcc -o hola exemples/hola.c
```
